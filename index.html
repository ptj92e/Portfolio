Got Jobs?

This is a webpage that I created with a team of 2 others. We used Bootstrap as our visual framework and features a fully responsive design. We modified it with our own class system for the styling. For the functionality, we used JQuery as well as a few API calls to retrieve information. We used The Muse, Teleport, and the Mapquest APIs for this app. We also used the third party API Plotly.js to create the table for the Teleport information.

Deployed: Got Jobs?

GitHub: Got Jobs?

The Ender Maze

The Ender Maze is a game developed using Handlebars.js and the ES6 class creation system to create characters and save them to a mySQL database. We used Sequelize as our ORM to hanle the connections to the database. For functionality and further development, we intend on switching the game over from Handlebars.js to React.js.

Deployed: The Ender Maze

GitHub: The Ender Maze

Weather Dashboard

I created this app as part of an assignment in my bootcamp. This app utilizes Bootstrap for a fully responsive mobile experience. For functionality, I used JQuery to send information to the Openweather API. The user can search for a city and the page will populate with current weather information and a 5 day forecast. When the page loads, the app will also use the user's geolocation to pull up weather information for where the user is located.

Deployed: Weather Dashboard

GitHub: Weather Dashboard

Profile Generator

This is a Node application that I created that uses Inquirer to prompt the user for their GitHub username and favorite color. Once those two answers are given, a PDF of information from GitHub is created and can be used as a quick resume reference.

Template Engine

This is another Node.js app that was an exercise in creating and modifying classes. The app prompts the user to add a new employee. From there, it asks the user to input the employee's name and email. If the employee is a manager, it asks for their office number. If the employee is an engineer, it asks for their GitHub username. If the employee is an intern, it asks for which school they went to.

Please visit GitHub to see application.

Employee Tracker

This is a command line CRUD app I developed which allows the user to create, read, update, and delete information in a mySQL server. Through a series of inquirer prompts, the user can create new departments, roles, and employees for a business, read all the information for the company, update an employee's information, and even delete employees, roles, and departments.

GitHub: Employee Tracker

Burger Logger

This is a full stack application utilizing Sequelize to manipulate a mySQL server to post and view a "burger wish list." This app uses a mySQL connection as well as Handlebars.js to populate the list of burgers on the screen. To be deployed on Heroku, this app also requires a connection to the JawsDB provided by Heroku. That connection is set using a config.json file that determines which connection is appropriate depending on the host.

GitHub: Burger Logger

Deployed on Heroku: Burger Logger
